### Konteks

VueTreez dibuat sebagai starter reusable frontend untuk menghilangkan boilerplate berulang dan memberi baseline stabil bagi pengembangan produk Vue modern.

### Solusi

- Struktur routing siap pakai dengan `MainLayout` serta route Home/About/Not Found.
- Internationalization bawaan dengan locale Inggris dan Indonesia serta pergantian bahasa yang cepat.
- Theme switching (`light`, `dark`, `auto`) dengan preferensi tersimpan dan perilaku UI yang konsisten.
- Persisted app state menggunakan Pinia + plugin storage dengan konvensi prefix key.
- Deployment production publik tersedia di `https://vuetreez.redwans.com/`.

### Catatan Arsitektur

- Setup Tailwind v4 berbasis CSS-first dengan token desain di `src/assets/css/main.css`.
- Segmentasi folder yang jelas (`components`, `layouts`, `pages`, `router`, `stores`, `i18n`) untuk maintainability.
- Script berbasis Bun untuk install/dev/build/preview dengan Oxlint + Prettier untuk quality gate.
