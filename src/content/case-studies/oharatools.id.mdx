### Konteks

Oharatools dibuat sebagai toolbox terfokus untuk kebutuhan developer sehari-hari
yang biasanya tersebar di banyak situs, browser extension, dan potongan CLI.

### Solusi

- Menggabungkan berbagai utilitas dalam satu homepage, termasuk Base64, URL
  encode/decode, JSON formatter, inspeksi JWT, regex tester, konversi timestamp,
  password generator, hashing, generator UUID/NanoID, SQL formatter, diff tool,
  dan cron parser.
- Mengurangi friction dengan pola navigasi cepat agar pengguna dapat berpindah
  antar tool tanpa perlu beradaptasi ulang dengan antarmuka baru.
- Menjaga pengalaman tetap langsung bisa dipakai dari browser melalui halaman
  utilitas khusus dan sitemap publik untuk discovery.

### Catatan Arsitektur

- Halaman tool diekspos sebagai route khusus seperti `/base64`,
  `/json-formatter`, `/jwt-tools`, `/diff`, dan `/cron-parser`.
- Homepage berperan sebagai layer registry untuk discovery utilitas dan titik
  masuk cepat ke workflow tertentu.
- Pemrosesan browser-first cocok untuk encoding, formatting, dan banyak helper
  kriptografi karena meminimalkan latensi bagi pengguna akhir.
